# Predict Customer Churn
For this project I use data from this [Kaggle](https://www.kaggle.com/competitions/playground-series-s6e3/overview) competition.

This is a classification problem. The aim is to predict whether a customer will churn. The prediction will be either 'yes' or 'no'.

## Setup
### Install uv
[installation instructions](https://docs.astral.sh/uv/getting-started/installation/)

### Install packages and create virtual environment
```bash
uv sync
```

### Get data
Download data from [Kaggle competition](https://www.kaggle.com/competitions/playground-series-s6e3/data).  
Save data to ``data/raw``
